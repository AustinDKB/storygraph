|---|---|

| `temperature_anomaly.csv` | HadCRUT5 (Met Office Hadley Centre) |
| `renewables_share.csv` | Ember Global Electricity Review 2024 |
| `gdp_vs_happiness.csv` | World Happiness Report 2024 |
| `life_expectancy_who.csv` | WHO Life Expectancy Dataset |
| `life_expectancy_gender.csv` | UN World Population Prospects 2024 |

Per the CC-BY 4.0 license, you must give appropriate credit when using this data. Each graph script includes a source note on the chart itself via `add_source_note()`.
